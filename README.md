**python*** is a high-level-general-purpose programming language.it is famous for its simple,English-like syntax.

 - **Easy to learn**:Its readable syntax which reduces the cost of program maintenace.

 - **Massive ecosystem**:it has thousands of free open-source libraries and frameworks that speed up development.

**JavaScript** is a high-level, versatile programming language mainly used to create interactive and dynamic web pages.
It runs in web browsers and can also be used on servers through environments like Node.js.

### Advantages of Using JavaScript
- **Easy to Learn**: It has a simple syntax and is beginner-friendly.
- **Interactive Websites**: It allows developers to create dynamic web content such as animations, forms, and games.
- **Cross-Platform**: It works on almost all modern web browsers and operating systems.

### Major Uses of JavaScript
- *Web Development*: Used to build interactive websites and web applications.
- *Mobile App Development*: Frameworks like React Native allow developers to create mobile apps.
- *Server-Side Development*: Node.js enables JavaScript to run on servers and handle backend operations.

### Rust
Rust is a multi-paradigm,open-source system programming language designed specifically foy high perfomance,type safety and memory security without relying on a garbage collector
### key features of rust
**memory safety without garbage collection**:Rust uses an innovative ownership and borrowing system.
**blazingly fast performance**:because it compiles directly down to native machine code using LLVM and lacks a runtime virtual machine.
**fearless concurrency**:managing multiple threads simultaneously is notoriously difficult
**Excellent developer toooling**:this language ships with its own robust ecosystem tool named cargo,which manages dependancies (known as crates),builds your projects and runs unit test seamlessly
